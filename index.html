Git Cheat Sheet

Every command you'll ever need, organized by category

🆕 Setup
git config --global user.name "Name"Set your name
git config --global user.email "email"Set your email
git initStart a new repo
git clone <url>Copy a repo
📸 Snapshot
git statusCheck what changed
git add .Stage all changes
git add <file>Stage a file
git commit -m "msg"Save a snapshot
git diffSee what changed
🌿 Branching
git branchList branches
git branch <name>Create branch
git switch <name>Go to branch
git switch -c <name>Create & switch
git branch -d <name>Delete branch
🔀 Merging
git merge <branch>Merge a branch
git rebase <branch>Rebase onto branch
git cherry-pick <hash>Pick one commit
git merge --abortCancel a merge
☁️ Remote
git remote -vShow remotes
git push origin mainUpload to GitHub
git pullDownload + merge
git fetchDownload only
⏪ Undo
git restore <file>Undo file change
git reset HEAD~1Undo last commit
git revert <hash>Safe undo commit
git stashSave work for later
git stash popRestore stashed work
🔍 Inspect
git logSee commit history
git log --onelineShort history
git show <hash>See a commit
git blame <file>Who wrote each line
🏷️ Tags & Releases
git tag v1.0Create a tag
git tagList all tags
git push --tagsPush tags to GitHub
git tag -d v1.0Delete a tag

Real World Examples

GitHub concepts explained with everyday analogies!

📚

Repository = School Bag

A repo is like your school bag — it holds all the books (files) for one subject (project). Your bag has a history of everything you ever put in it!

📸

Commit = Photo Album

Every commit is like taking a photo of your room. If it ever gets messy, you can look at old photos and go back to when it was clean!

🛣️

Branch = Parallel Story

Imagine writing a story. A branch lets you write an alternate ending — if it's good, you add it to the main story. If not, you throw it away!

🍕

Fork = Copy a Recipe

Forking is like copying your friend's pizza recipe into your own notebook, so you can change the toppings without changing theirs!

✉️

Pull Request = Suggestion Letter

A PR is like writing a letter to your teacher saying "I improved the homework! Please check and add it to the class notes!"

⚔️

Merge Conflict = Two Cooks

Two people changed the same line — like two cooks adding different amounts of salt. You have to decide which amount (or combination) to keep!
